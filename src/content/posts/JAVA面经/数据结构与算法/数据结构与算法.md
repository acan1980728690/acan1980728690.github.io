## 总纲领

种种数据结构，皆为**数组（顺序存储）**和**链表（链式存储**）的变换。

数据结构的关键点在于**遍历**和**访问**，即增删查改等基本操作。

种种算法，皆为**穷举**。

穷举的关键点在于**无遗漏**和**无冗余**。熟练掌握算法框架，可以做到无遗漏；充分利用信息，可以做到无冗余。

##### 求二叉树的最大深度

```java
int maxDeath(TreeNode node){
    if(node==null){
        return 0;
    }
    int left = maxDeath(node.left);
    int right = maxDeath(node.right);
    return Math.max(left,right) + 1;
}
```

int getMin(TreeNode root){
    if(root == null){
        return Integer.MAX_VALUE;
    }
    if(root.left == null&&root.right == null){
        return 1;
    }

##### 求二叉树中第k层节点的个数

```java

int numsOfkLevelTreeNode(TreeNode root,int k){
    if(root == null||k<1){
        return 0;
    }
    if(k==1){
        return 1;
    }
    int numsLeft = numsOfkLevelTreeNode(root.left,k-1);
    int numsRight = numsOfkLevelTreeNode(root.right,k-1);
    return numsLeft + numsRight;
}
```

##### 求两个二叉树的最低公共祖先节点

```java
TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
    // 边界条件
    if (root == null || root == p || root == q) {
        return root;
    }
    
    // 在左右子树中查找
    TreeNode left = lowestCommonAncestor(root.left, p, q);
    TreeNode right = lowestCommonAncestor(root.right, p, q);
    
    // 如果左右都找到，当前root就是LCA
    if (left != null && right != null) {
        return root;
    }
    
    // 否则返回非空的那一侧
    return left != null ? left : right;
}
```

##### 层次遍历

记住这两个模板

```java
void levelOrderTraverse(TreeNode root) {
    if (root == null) {
        return;
    }
    Queue<TreeNode> q = new LinkedList<>();
    q.offer(root);
    while (!q.isEmpty()) {
        TreeNode cur = q.poll();
        // 访问 cur 节点
        System.out.println(cur.val);

        // 把 cur 的左右子节点加入队列
        if (cur.left != null) {
            q.offer(cur.left);
        }
        if (cur.right != null) {
            q.offer(cur.right);
        }
    }
}

```

```java
void levelOrderTraverse(TreeNode root) {
    if (root == null) {
        return;
    }
    Queue<TreeNode> q = new LinkedList<>();
    q.offer(root);
    // 记录当前遍历到的层数（根节点视为第 1 层）
    int depth = 1;

    while (!q.isEmpty()) {
        int sz = q.size();
        for (int i = 0; i < sz; i++) {
            TreeNode cur = q.poll();
            // 访问 cur 节点，同时知道它所在的层数
            System.out.println("depth = " + depth + ", val = " + cur.val);

            // 把 cur 的左右子节点加入队列
            if (cur.left != null) {
                q.offer(cur.left);
            }
            if (cur.right != null) {
                q.offer(cur.right);
            }
        }
        depth++;
    }
}
```

**b 树和 b+ 树**

B树是一种多叉的、自平衡的搜索树，拥有以下性质

通常我们说m阶的B树，它必须满足如下条件： 

1. 每个非叶子节点最多只有m个子节点。
2. 每个非叶子节点（除了根）具有至少⌈ m/2⌉子节点
3. 如果根不是叶节点，则根至少有两个子节点。
4. 具有k个子节点的非叶节点包含k -1个键。
5. 所有叶子结点高度一致。

在B树中，任何一种节点都会存储数据，并且该数据只会出现一次，因此搜索可能在非叶子节点就结束了，搜索的性能相当于做一次二分查找

B+ 树

B+ 树和 B 树的主要区别是：

1. 所有数据存储在叶子节点
2. 非叶子节点都可以视为索引
3. 所有的叶子节点会形成一个有序链表
