主要是一种简单的用来传输数据的格式，轻量级、易于阅读和编写。特点是都有""包裹的键值对，键和值之间用冒号:分隔，键值对之间用逗号,分隔。

**简单示例**

```

{
    "name":"aaa",
    "age":18,
    "isStudent":true,
    "hobbies":["reading","swimming"],
    "address":{
        "street":"xxx", 
        "city":"yyy",
        "state":"zzz"
        }
    

}
```
